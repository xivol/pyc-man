# pyc-man
Pygame PAC MAN

Simple [PAC-MAN](https://en.wikipedia.org/wiki/Pac-Man) clone.
 - Use arrows to move.
 - Eat Energizers to scare ghosts.
 - Ghosts speed up with each level.
 - Dumb AI, only Blinky :(
